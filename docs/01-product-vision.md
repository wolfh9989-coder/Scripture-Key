# Product Vision: Scripture Key

## Mission

Scripture Key transforms Bible study from static reading into an interactive, explainable, and connected intelligence experience.

## Positioning

Scripture Key is:

- Bible + Knowledge Graph (Lattice)
- Bible + Pattern Intelligence (UMIE)
- Bible + Credibility-Scored Community Layer (Artifact)
- Bible + Voice Command Interface (BRCIS)

## Signature Value

Most Bible apps provide:

- Search
- Commentary lookup
- Reading plans

Scripture Key provides:

- Dynamic verse expansion through layered interpretation
- Inter-verse graph exploration by theme, person, event, and pattern
- Explainable intelligence outputs with confidence and supporting evidence
- Conversation-driven Scripture analysis through voice and text

## User Modes

1. Beginner: simple explanations, low jargon, practical application
2. Study: deeper textual context, historical background, cross references
3. Deep/Theological: language nuance, interpretive alternatives, doctrinal frameworks
4. Devotional: reflection guidance, prayer prompts, practical spiritual action

## Product Non-Negotiables

- Every generated insight must include evidence references.
- Alternative interpretations must be surfaced when confidence is mixed.
- Weak or speculative claims must be clearly labeled.
- User-generated insights require credibility scoring and moderation flow.
- The platform must support both spiritual and analytical engagement.

## Strategic Advantage

Scripture Key is not just content delivery. It is a living network that unifies text, structure, theology, and intelligence in one interface.
