# Phased Roadmap

## Phase 1: Core Study Engine

Objective: Deliver a high-quality interactive Scripture viewer with foundational intelligence.

Scope:

- Scripture Viewer with split-screen layout (Verse | Commentary | Original Text)
- Cross references and Strong's definitions
- Key Mode unlock for verse layers:
  - Literal
  - Historical
  - Prophetic
  - Symbolic
  - Related verses
- Basic Lattice connections for themes, people, events

Exit Criteria:

- User can open any verse and view layered expansion with traceable sources.
- Graph-backed related verse suggestions are available in under 1.5s median.

## Phase 2: Intelligence and Conversation

Objective: Make Scripture study conversational and pattern-aware.

Scope:

- BRCIS voice and text command parsing
- Complex comparative queries:
  - Compare chapter pairs
  - Find all sayings by entity/topic
- UMIE pattern engine:
  - Repeated phrases
  - Numeric motifs (7, 12, 40)
  - Basic symmetry/chiastic candidates
- Timeline view for events and fulfillment links

Exit Criteria:

- Voice command success rate reaches target intent parse accuracy.
- Pattern outputs include confidence scores and evidence spans.

## Phase 3: Community and Trust

Objective: Add discussion and insight credibility systems.

Scope:

- Verse-linked user insights
- Credibility scoring model
- Verified insight badges with moderation policy
- Reputation and abuse controls
- Insight feed per verse/theme/node

Exit Criteria:

- Insight quality controls are active with clear transparency markers.
- Misinformation handling workflow is operational.

## Phase 4: Advanced Exploration

Objective: Launch high-impact differentiators.

Scope:

- Prophecy Tracker (fulfilled/unfulfilled mapping)
- DNA of Scripture analytics (frequency and structural encoding)
- Sermon Builder Mode (outline, references, application)
- Exploration Mode (guided discovery and gamified unlocking)

Exit Criteria:

- Each advanced module has measurable usage and study outcome metrics.
