# UI and UX Direction

## Visual North Star

Scripture Key should feel like Scripture in a living digital cosmos.

Design language:
- Dark, high-contrast base with warm accent highlights
- Glowing graph nodes and flowing connection lines
- Layered depth and atmospheric backgrounds
- Typographic hierarchy that distinguishes Scripture text from analysis text

## Core Screens

1. Study Cockpit
- Left pane: Scripture text and quick navigation
- Center pane: Unlock layers and explanation stack
- Right pane: graph/timeline/map tabs

2. Lattice Explorer
- Force-directed graph with filter chips
- Node detail drawer with evidence and link types

3. Pattern Lab
- Pattern filters and confidence slider
- Structural visualizations for symmetry and repetition

4. Insight Commons
- Verse-linked discussion with credibility markers
- Verified badges and moderation state labels

## Interaction Principles

- One-click verse unlock
- Instant context reveal without page transitions
- Progressive disclosure for deeper layers
- All generated claims must have accessible references

## Motion Guidance

- Soft load-in for verse panels
- Staggered reveals for unlock layers
- Subtle edge-flow animations on graph connections
- Reduced motion mode for accessibility

## Accessibility Baseline

- WCAG AA contrast targets
- Keyboard navigation across all panes
- Screen-reader labels on graph and timeline controls
- Adjustable text size and reading density

## Mobile Adaptation

- Bottom tab layout for Verse, Unlock, Graph, Insight
- Swipe between layers
- Sticky verse header with quick reference context
